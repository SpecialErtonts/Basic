Shell нь Линукс компьютерийг удирдахад зориулагдсан текстэд суурилсан интерфейс юм. Microsoft-ын Powershell эсвэл cmd-тэй адил энэ нь GUI-ээс гадна хэрэглэгч болон цөмийн хоорондох интерфейс юм. (Graphical User Interface).

Янз бүрийн төрлийн shell байдаг бөгөөд тус бүрийг өмнөх хувилбарууд дээр үндэслэн сайжруулсан эсвэл тодорхой зорилгод зориулж оновчтой болгосон.

Shell нь компьютерт зааврыг хүргэх хамгийн хурдан бөгөөд үр дүнтэй арга учраас хакерууд ихээр ашигладаг. GUI нь сайн, гэхдээ зарим функцэд графикаар хандах боломжгүй, эсвэл таны ашиглахыг хүссэн хэрэгсэлд график интерфейс байхгүй тул хязгаарлагдмал байж болно..

Зарим нийтлэг Shell орно:

The Bourne shell (sh)
The GNU-Bourne Again shell (bash)
The Z shell (zsh)
The C shell (csh)
The Korn shell (ksh)
Шуурхай хичээл: "Терминал" ба "бүрхүүл" гэсэн үгсийг кибер аюулгүй байдлын ертөнцөд болон энэ нийтлэлийн туршид сольж хэрэглэдэг. Гэхдээ, тэд өөр. Терминал нь график интерфейсээр дамжуулан бүрхүүлд нэвтрэх боломжийг олгодог програм юм.
